Field State Color-Coding System Test

Field State Legend

Error State: Pale red with dark red outline - Field has validation errors
Required Empty: Pale yellow with dark yellow outline - Required field not filled
Optional Empty: Pale green with dark green outline - Optional field with no data
Filled/Valid: No color (plain white) - Field properly filled out
Disabled: Light gray with greyed text - Field cannot be edited
